GET - /api/authenticate

GET /api/authenticate

Endpoint de testes, apenas recebe um token e confirma que o usuário está autenticado.

Parâmetros

Nenhum.

Headers

Atributo Valor
Authorization Bearer <token>

Substitua <token> pelo valor recebido em POST - /api/authenticate.
